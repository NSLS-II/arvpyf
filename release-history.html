v0.4.1 (2019-06-05)¶

v0.4.0 (2019-05-05)¶

Archiver Python Frontend extended with the ArchiverReader class for providing an interface to the EPICS Archiver Appliance Retrieval service.

ArchiverConfig interface updated with the get_pv_details and change_archival_parameters methods.

v0.2.0 (2018-08-19)¶

ArchiverConfig interface updated with the get_pv_type_info and resume_archiving_pvs methods and added arguments to archive_pvs and delete_pv.
